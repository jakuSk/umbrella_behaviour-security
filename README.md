#### Load umbrella key and secret stored in sysenv

Method get_secrets to get secrets for cisco Umbrella
```python
def get_secrets(key, secret) -> (str, str):
    try:
        key = os.environ[key]
        secret = os.environ[secret]

        return key, secret
    except Exception as e:
        print("Error: {}".format(e))
```
